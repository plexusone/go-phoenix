## [0.1.0] - 2026-01-04

### Added

- OpenTelemetry integration (`otel` package) with OTLP HTTP exporter ([`bcd0e2f`](https://github.com/plexusone/phoenix-go/commit/bcd0e2f))
- Phoenix Cloud support via `PHOENIX_API_KEY` and `PHOENIX_SPACE_ID` environment variables ([`bcd0e2f`](https://github.com/plexusone/phoenix-go/commit/bcd0e2f))
- OpenInference semantic conventions for LLM observability attributes ([`bcd0e2f`](https://github.com/plexusone/phoenix-go/commit/bcd0e2f))
- Batch and sync span processing modes ([`bcd0e2f`](https://github.com/plexusone/phoenix-go/commit/bcd0e2f))
- REST API client for Projects, Spans, Datasets, Experiments, Prompts, Annotations ([`bcd0e2f`](https://github.com/plexusone/phoenix-go/commit/bcd0e2f))
